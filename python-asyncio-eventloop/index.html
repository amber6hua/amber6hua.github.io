# 个人理解总结

python async 异步 需要通过 loop.run_until_complete 启动
默认主线程有 eventLoop，可以直接 get_event_loop
其他线程执行 async 异步，需要新创建 eventLoop, new_event_loop 后才能获取 get_event_loop，不然会提示错误
python3.7 以后， asyncio.run 以更简洁代码的启动 async
