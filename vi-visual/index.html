# 解决方法

首先寻找 vim 的配置文件：

find / -name "defaults.vim"

/usr/share/vim/vim82/defaults.vim

可以看到第一个结果是在 vim 目录下，所以是 vim 的配置文件，对文件内容进行编辑

vim /usr/share/vim/vim82/defaults.vim

·········
if has('mouse')
  if &term =~ 'xterm'
    set mouse=a  #找到这行，将等号右边的值改为“r”
  else
    set mouse=nvi
  endif
endif
